Report for compleat-fitness

  • Lots of Google searches for almost everything involved with the site. Lots of useful demos found on Stack Overflow and W3 Schools which were taken and adapted for all sorts of features.

General

  • (Beautiful) Nav panel uses only HTML and CSS. This was first taken from a codepen demo however they used FontAwesome for the icons. We adapted this so it uses custom made SVGs which were all made in Inkscape except for the cutlery icon The CSS has been extensively tied up and the styling and positioning changed. This took a long time to perfect but looks good now it's done.
  • Whole site has fluid design and everything is still readable no matter what size the window is made. Not tested on mobile.
  • 'Compleat Fitness' logo and favicon created from scratch using Gimp and royalty-free font 'Pecita'.
  • Inclusion of a favicon.

index.html:

  • Multiple different background images tried and tested against theme colours. Gimp was used to make copies with varying levels of opacity. The end result was mostly transparent (10% opacity), improving the legibility of overlying text.

packages.html:

  • Learned to write accordion from inspirationalpixels. Styled to our needs and work well for FAQs.
  • BMI Calculator uses JavaScript to take values from weight and height sliders, calculates BMI, and replaces the innerHTML of a DOM to display result. This happens on each change of sliders thanks to jQuery. Would like to add a Imperial/Metric toggle functionality if time after server side.

nutrition.html:

  • Using Instafeed to pull in pictures from Instagram was not too difficult, however we did have to set up an Instagram developer account and register the application. We think this plugin may be slightly broken in the respect that the pictures apparently cannot be resized. Connor is working on the plugin on his own branch on Github to try to reach a fix.
  • More difficult, managed to get the Instagram pictures in to a nice image slider Slick. This now works well and will be more effective when the images are larger.

contact.html:

  • Contact form elements looked up on W3Schools and styling from different sources searching online to see what could look good on our site.
  • Google map instructions taken from the Google Map API intro page.
